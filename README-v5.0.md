# 3C Studio v5.0 — Records + Scoreboard 整合版

## 核心改變
- Scoreboard 已整合為 `scoreboard.html`
- 所有頁面都在同一個 GitHub Pages Repository 與同一個 PWA
- 比賽結束直接寫入 `threeCushionTrainingLog_v1`
- 不再需要跨網站同步、網址橋接或等待另一個 App 更新
- 四個功能按鈕依擊球方排列：
  - 白球：暫停｜換黃球 →｜上一步｜結束比賽
  - 黃球：暫停｜上一步｜← 換白球｜結束比賽

## 部署
將 ZIP 內全部檔案上傳覆蓋 `3C-records` Repository 根目錄。
舊的 Scoreboard Repository 可保留但不再使用。

## iPhone / iPad
部署後請刪除舊的兩個主畫面捷徑，再從整合後的 3C Studio 首頁重新「加入主畫面」。
