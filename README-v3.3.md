# 3-Cushion Training System v3.3

## Duel scoring
- Single match and formal match modes now track both players.
- The app alternates between self and opponent after each completed run.
- Each player has individual score, innings, AVG, and High Run.
- History stores scoreline and result: win, loss, or draw.
- Analytics includes match count, win rate, W/L/D, and average score margin.
