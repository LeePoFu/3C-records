# 3-Cushion Training System v3.2

## Fixes
- All activity-mode controls now share consistent dimensions, spacing, radius, and full-width behavior.
- Multiplayer top summary now shows the current player's own score, innings, AVG, and High Run.
- Multiplayer records no longer store misleading combined totals as the top-level game result.
- Participant summary highlights the active player.
- History and dashboard show multiplayer games as individual-stat sessions rather than combined AVG.
