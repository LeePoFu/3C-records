# v4.3.2 下一位與本杆時間修正版

## Bug 修正
- 修正多人追分「下一位」按鈕完全無反應。
- 原因是 v4.3.1 誤呼叫不存在的 `pushQuickUndoState()`。
- 改回正確的 `pushQuickUndo()`，並維持先更新分數、再切換球員、最後重繪。

## 時間定義整理
- 即時計時器：單次擊球時間，每次操作後歸零。
- 表格每列：一個 inning／整杆。
- 表格欄位改為「本杆時間」。
- 本杆時間從該球員開始本杆，到按下下一杆、下一位或換對手為止。
- 暫停期間不計入本杆時間。
- 單次擊球時間仍保存在 Rhythm Analytics 與逐次操作紀錄。
